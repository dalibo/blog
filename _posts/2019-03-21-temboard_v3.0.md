## Changelog

    Affichage en plein écran dès la page d'accueil et le tableau de bord
    ? [Limit double authentication to not read only APIs]
    Plugin de maintenance
    Barre latérale glissable [Collapsible sidebar]
    Délai de copie et de connexion, fichiers temporaires [New monitoring probes: replication lag and connection, temporary   files]
    Tests de fonctionnalité de l'IU
    Support Tornado 4.4 and 5
    Script d'auto-configuration
    Affichage des requêtes en attente ou bloquées dans la barre d'activité [Show number of waiting/blocking req in activity tabs]
    Affichage du statut de disponibilité sur la page d'accueil
    Tableau de bord comme la page d'accueil [Dashboard like home page]
    Amélioration des vues de l'activité [Improve activity views]
    Révision du framework web [Review web framework]
    Mise à jour des paquets Debian [Review debian packaging]
    Suppression de pg_hba.conf et pg_ident.conf edition du plugin pgconf
    Correction du traitement des données de contrôle qui ne sont plus coincées dans la file d'envoi [Avoid monitoring data to get stuck in agent sending queue]
    Correction et mise à jour de la documentation [Documentation cleaning and updates]
    Réduction des commandes rollback inutiles dans les requêtes de lecture (base de données du dépôt) [Limit useless rollback statements on read only queries (repository database)]
